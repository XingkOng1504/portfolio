# Portfolio BUT Informatique S4 — Lin Huang Christophe

Portfolio de fin de BUT Informatique S4, IUT Villetaneuse (Sorbonne Paris Nord).

## Déploiement GitHub Pages

1. Créer un dépôt GitHub nommé `portfolio-s4` (ou `<ton-username>.github.io` pour l'URL principale)
2. Pousser ce dossier sur la branche `main`
3. Aller dans **Settings → Pages → Source → Deploy from branch → main / root**
4. Le site sera disponible à `https://<ton-username>.github.io/portfolio-s4/`

## Structure

```
portfolio-github/
├── index.html    ← page principale
├── style.css     ← styles
├── script.js     ← navigation active
└── README.md
```
